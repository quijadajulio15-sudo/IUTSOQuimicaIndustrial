El Verbo "To Be" — Clase Rápida

Clase rápida: El verbo "to be" se usa para identificar (I am a technician), describir estados (The solution is hot) y ubicar (They are in the lab). Formas principales: presente (am/is/are) y pasado (was/were). Para negar, añade "not" o contracciones (isn't/aren't/wasn't/weren't).

Presente — Afirmativo (Clase)

Estructura: sujeto + am/is/are. Ejemplos: I am a chemist. You are in the lab. It is reactive.

Presente — Negativo (Clase)

Estructura: sujeto + am/is/are + not. Ejemplo: The machine isn't calibrated.

Pasado — Afirmativo (Clase)

Estructura: sujeto + was/were. Ej.: They were on duty yesterday.

Pasado — Negativo (Clase)

Estructura: sujeto + was/were + not. Ej.: The system wasn't online.

Verbos Modales — Clase Rápida

Breve: Los verbos modales expresan permiso (may/can), obligación (must), recomendación (should) y posibilidad (might). Ejemplos: "You must wear PPE" (obligación), "You can use the fume hood" (permiso/habilidad).

Gerundio (-ING) — Clase Rápida

Breve: El gerundio se usa para acciones en progreso (He is mixing), como sujeto (Mixing reagents is important) y tras preposiciones (before adding). Usa la forma base + -ing.

Pronombres Posesivos — Clase Rápida

Breve: Indican posesión: my/your/his/her/its/our/their; y pronombres suyos: mine/yours/his/hers/ours/theirs. Ej.: "This is our lab", "The report is theirs".
